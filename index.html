⚠️ Most common reason it doesn't work:
When deploying, Who has access must be set to Anyone — not "Anyone with a Google account". If you've already deployed, click Deploy → Manage deployments → edit the pencil → change access → re-deploy a new version.
One-time setup (5 min):
1. Open script.google.com → New project → delete all default code
2. Tap 📋 Copy Script Code below → paste into the editor → Save (Ctrl+S)
3. Click Deploy → New deployment
4. Type: Web app · Execute as: Me · Who has access: Anyone (not "with Google Account")
5. Click Deploy → copy the Web App URL → paste above → Save & Test
6. If you change the script later, choose Deploy → Manage deployments → New version
How to get your credentials
1. Create your bot
Open Telegram, search @BotFather, send /newbot, follow the steps. Copy the token it gives you.

2. Get your Chat ID
Search @userinfobot on Telegram, send any message. It replies with your numeric ID — paste that above.

3. Customer Chat IDs
Customers must send /start to your bot first. Ask them to DM you their Telegram username so you can look up their chat ID via your bot's getUpdates endpoint. This enables Flow 3 (ready alerts to customer).
